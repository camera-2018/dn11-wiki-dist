Hello DN11 ​

DN11 全称为 “11 号去中心化网络”（Decentralized Network 11），得名于其发源地杭电 11 楼，不过也接纳来自其他宿舍楼其他院校的同志。

在 DN11 您可以和一群热爱计算机网络的同学们一起，从两手空空开始购买设备并重新搭建互联网，一起重走互联网发展路。

您可以亲手从隧道开始，逐步配置各种路由协议，搭建属于您的全球网络；您可以亲自回到 Web1 的早期，从 DNS 开始，搭建自己的 Web 服务，并通过门户页，搜索引擎重新建立整个网络的联系；在未来我们也会一起探索在 DN11 重新发展 Web2、Web3 的可能性。

当然 DN11 是自由的，计算机网络的世界是宽广的。DN11 并不局限于 Web 内容的发展。

如果您对通信以及硬件感兴趣，我们也鼓励您从最简单的八木天线开始制作属于自己的物理层，同时也有机会给他们定制一个链路层；

如果您对路由交换感兴趣，我们也可以一起研究内部外部的各种路由协议和 Peer 方式，一起思考各种选路策略给整个网络带来的正面以及负面影响，竭尽全力在复杂的网络环境下优化 DN11 的整体连通性；

如果您对隧道和代理感兴趣，我们也欢迎您来一起讨论各种协议之间的区别，您还有机会同我们一道编写各种高性能或场景定制化隧道代理协议；

如果您对网络开发等方向感兴趣，我们也欢迎您来一起将 DN11 内的各种需求进行集成以及开发，包括但不限于自动打洞、AutoPeer、快速 DDNS...;

...
